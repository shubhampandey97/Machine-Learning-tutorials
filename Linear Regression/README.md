# 📈 Linear Regression

This module covers the fundamentals and advanced concepts of **Linear Regression**, one of the most important supervised learning algorithms in machine learning.

It includes both **mathematical understanding** and **hands-on implementation** using Python.

---

# 🎯 What is Linear Regression?

Linear Regression is a supervised learning algorithm used to model the relationship between:

* Independent variables (features)
* Dependent variable (target)

It assumes a **linear relationship** between input and output.
